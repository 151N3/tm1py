TM1py : 
=======================

TMpy is a object oriented interface to the new IBM Cognos TM1 REST API.

.. code-block:: python

    >>> r = requests.get('https://api.github.com', auth=('user', 'pass'))
    >>> r.status_code
    204
    >>> r.headers['content-type']
    'application/json'
    >>> r.text
    ...
